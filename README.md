# Auction Website Project
## Project Overview
A modern, responsive auction website called "Оно тебе надо" (translates to "You Need It") featuring unique items that nobody believed in. The website showcases various auction lots with artistic design elements and clean typography.
## HTML Structure
- Semantic HTML5 elements (<header>, <main>, <section>, <footer>)
- Proper use of <address> element for contact information
- Accessible navigation with ARIA-friendly markup
## CSS Features
- Custom Fonts: Raleway Regular and Bold weights with WOFF2/WOFF formats
- CSS Grid & Flexbox: For complex layouts and alignment
- Overlay Effects: Using pseudo-elements for image darkening
- Typography Control: Custom letter spacing and line heights
